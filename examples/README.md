# ESP Chas TV Examples

This directory contains example code and projects for the ESP Chas TV collaboration.

## Available Examples

### 1. Basic LED Control
Simple example showing how to control LEDs for TV show effects.

### 2. WiFi Remote Control
Example of controlling the show via WiFi and web interface.

### 3. Multi-Device Sync
Synchronizing multiple ESP32 devices for coordinated effects.

### 4. Sensor Integration
Using sensors to create interactive show elements.

### 5. Display Integration
Working with various displays (OLED, TFT, LED Matrix).

## How to Use Examples

1. Copy the example to your `src/main.cpp` file
2. Install any required libraries
3. Adjust configuration for your hardware
4. Build and upload: `pio run --target upload`

## Contributing Examples

Have a cool example to share? 
1. Create a new directory with your example name
2. Include complete code and a README
3. Document required hardware and libraries
4. Submit a pull request!
